# **useLocaleNotifications**

### Provides a direct interface to trigger browser system notifications within a component.

#### **Installation**

<CodeBlock
  code={`npm install @vibehooks/react/useLocaleNotifications`}
  language="bash"
/>

#### **Description**

This hook simplifies the native Browser Notification API by managing the permission state internally and providing a clean functional interface to trigger alerts without managing global stores.
